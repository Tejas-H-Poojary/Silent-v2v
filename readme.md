<strong>SafeSphere Route — Project Summary</strong>
<strong>What it is</strong>

A hackathon project for women's personal safety, tagline: "See which street is lit before you walk it."
A browser-based safe-route mapping web app that helps users pick well-lit, lower-risk walking routes instead of just the fastest one.

<strong>Core concept</strong>

User enters a From and To location; the app computes multiple walking routes and scores each one for safety, not just speed/distance.
Safety score blends three signals: real-time "eyes on the street" (nearby business/foot-traffic density), community-submitted incident reports (harassment, poor lighting, felt unsafe, suspicious activity), and time-of-day.
Users can drop a pin on the map to report an incident, which visibly lowers the score of any nearby route in real time.

<strong>Key features</strong>

1)Custom From/To autocomplete with live suggestions
2)Multiple alternative walking routes, each with a color-coded safety score, distance, and duration
3)Tap-to-report incident flow with categorized markers on the map
4)Night/day theme toggle
5)"How scores work" explainer panel
6)Recent reports feed
